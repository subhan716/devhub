# DevHub Libraries & Packages

This file documents all external libraries, frameworks, and packages installed in the project to maintain a clean tracking record.

## Frontend (React + Vite)
| Package Name | Purpose |
|--------------|---------|
| `react` & `react-dom` | Core UI library |
| `react-router-dom` | Client-side routing for multi-page SPA |
| `tailwindcss` | Utility-first CSS styling framework |
| `framer-motion` | Complex animations and gestures |
| `lucide-react` | Clean and professional SVG icon set |
| `axios` | HTTP client for making API requests to the backend |
| `react-hot-toast` | Professional popup notifications |
| `three` | 3D graphics library for WebGL |
| `@react-three/fiber` | React renderer for Three.js |
| `@react-three/drei` | Useful helpers for react-three-fiber |
| `tailwind-scrollbar` | Custom cross-browser scrollbar styling |
| `react-syntax-highlighter` | Syntax highlighting for code snippets in posts |
| `lenis` | High-performance buttery smooth scrolling library |

## Backend (Node.js + Express)
| Package Name | Purpose |
|--------------|---------|
| `express` | Core web server framework |
| `mongoose` | MongoDB object modeling and database interaction |
| `dotenv` | Environment variable management |
| `cors` | Cross-Origin Resource Sharing for frontend/backend comms |
| `bcryptjs` | Secure password hashing |
| `jsonwebtoken` | JWT generation for authentication (OAuth 2.0 flow) |
| `nodemon` | Development server auto-reloading |
